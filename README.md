# SURVIVING IN UNDERGROUND

A 2D Java Swing arcade survival game where players collect gems, avoid obstacles, defeat enemies, and survive as long as possible.

## Overview

The game is a fast-paced desktop game built with Java and Swing. The player controls a character that can move, jump, and shoot while collecting valuable gems and avoiding dangerous obstacles.

The objective is to score as many points as possible before losing all lives or reaching the end of the survival timer.


## Features

- Character movement system
- Jump mechanics with gravity
- Animated character sprites
- Projectile shooting system
- Collectible gems
- Hazardous falling rocks
- Score tracking
- Life system
- Survival timer
- Multiple difficulty levels
- Pause and game-over screens
- Custom graphics and animations


## Controls

| Key | Action |
|------|----------|
| A | Move Left |
| D | Move Right |
| SPACE | Jump |
| ENTER | Shoot |


## Gameplay

### Collect Gems

Collect falling gems to earn points.

- Gem = +10 points

### Avoid Rocks

Avoid dangerous falling rocks.

- Rock = -10 points

### Defeat Enemies

Avoid or shoot enemies to survive.

### Survive

The goal is to survive for 60 seconds while achieving the highest possible score.
